## 添加/替换图片

### 替换现有图片

1. 打开对应分类文件夹（如 `images/foxiang/`）
2. 把图片命名为 `1.png`, `2.png`, `3.png`...
3. 复制进去覆盖原文件
4. 刷新网页即可

### 增加产品数量

1. 在对应文件夹添加图片（如 `16.png`, `17.png`）
2. 编辑 `script.js`，找到 `categories` 数组
3. 修改对应分类的 `imageCount` 数量
4. 刷新网页

## 图片要求

| 属性 | 要求 |
|------|------|
| 格式 | PNG 或 JPG |
| 尺寸 | 600x400 像素（或相同比例） |
| 大小 | 建议 < 500KB |
| 命名 | 从 1 开始的连续数字 |
| 方向 | 建议横向（风景） |
