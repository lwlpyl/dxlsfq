# 背景音乐

## 使用方法

1. 准备 MP3 格式音乐文件
2. 重命名为 `bg.mp3`
3. 放入此文件夹
4. 刷新网页

## 推荐音乐类型

- 古琴曲
- 佛教音乐
- 禅意音乐
- 冥想音乐
- 轻音乐

## 注意事项

- **浏览器限制**: 首次播放需要用户点击页面任意位置
- **文件格式**: 仅支持 MP3
- **文件大小**: 建议 < 5MB
- **音量**: 初始音量 30%，可在 `script.js` 中调整

## 如何控制

- 点击右下角音乐按钮：播放/暂停
- 页面加载后自动尝试播放
- 任何用户交互（点击/滚动/按键）触发播放
