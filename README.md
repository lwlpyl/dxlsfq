## 图片管理

### 分类配置

编辑 `script.js` 中的 `categories` 数组：

```javascript
const categories = [
  { id: "foxiang", name: "佛像", imageCount: 15 },
  { id: "tongzhong", name: "铜钟", imageCount: 4 },
  // ... 更多分类
];
```

### 添加产品图片

1. 在对应分类文件夹添加图片（如 `images/foxiang/`）
2. 按顺序命名：`1.png`, `2.png`, `3.png`...
3. 修改 `imageCount` 为实际数量
4. 刷新网页即可

### 图片要求

- **格式**: PNG 或 JPG
- **尺寸**: 建议 600x400 像素（或相同比例）
- **大小**: 建议 < 500KB
- **命名**: 从 1 开始的连续数字

## 背景音乐

### 使用方法

1. 准备 MP3 格式音乐文件
2. 放入 `audio/` 文件夹
3. 重命名为 `bg.mp3`
4. 刷新网页

### 控制方式

- 点击右下角 🎵 按钮：播放/暂停
- 页面加载后自动尝试播放
- 任何用户交互（点击/滚动）触发播放

## 使用技巧

### 键盘快捷键

- **←** : 上一张图片
- **→** : 下一张图片
- **Esc** : 关闭大图模式

### 图片浏览

1. 点击任意产品图片 → 放大查看
2. 点击 ❮ 或 ❯ → 切换图片
3. 点击 × 或背景 → 关闭

## 常见问题

### Q: 图片不显示？

A: 检查图片路径和文件名是否正确，确保从 1 开始连续命名。

### Q: 音乐不播放？

A: 浏览器限制自动播放，点击页面任意位置或点击 🎵 按钮。

### Q: 如何添加新分类？

A: 在 `categories` 数组添加新分类，创建对应文件夹，放入图片。
