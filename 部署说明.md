# 🚀 部署步骤

由于 GitHub 需要认证，请按以下步骤手动部署：

## 方法 1: 网页上传（最简单）

### 步骤 1: 创建仓库

1. 访问 https://github.com/new
2. Repository name: `dxlsfq`
3. Description: `李释佛器产品展示网站`
4. 选择 **Public**
5. **不要** 勾选 "Initialize this repository with a README"
6. 点击 "Create repository"

### 步骤 2: 上传文件

1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 打开 `D:\workspace\dxlsfq` 文件夹
3. 选择所有文件和文件夹（Ctrl+A）
4. 拖拽到 GitHub 上传区域
5. 等待上传完成
6. Commit message: `Initial commit: 李释佛器产品展示网站`
7. 点击 "Commit changes"

### 步骤 3: 启用 GitHub Pages

1. 进入仓库的 **Settings**（顶部导航）
2. 左侧菜单点击 **Pages**
3. **Source** 选择：`Deploy from a branch`
4. **Branch** 选择：`main` 和 `/ (root)`
5. 点击 **Save**
6. 等待 2-5 分钟

### 步骤 4: 访问网站

页面顶部会显示访问地址：
```
https://lwlpyl.github.io/dxlsfq/
```

点击即可访问！

## 🔧 遇到问题？

### 问题 1: 404 错误
**解决**: 等待几分钟，GitHub Pages 需要时间构建

### 问题 2: 图片不显示
**解决**: 
- 检查文件名大小写（GitHub 区分大小写）
- 清除浏览器缓存（Ctrl+Shift+Delete）

### 问题 3: 样式错乱
**解决**: 
- 强制刷新页面（Ctrl+F5）
- 检查 styles.css 是否正确上传

### 问题 4: 音乐不播放
**解决**: 
- 点击页面任意位置
- 或点击右下角音乐按钮
